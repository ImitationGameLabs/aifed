# aifed-common

Shared config, types, and error definitions for the [aifed](https://crates.io/crates/aifed) editor and [aifed-daemon](https://crates.io/crates/aifed-daemon).

This library crate holds the workspace/config parsing, hashline types, and error enums shared between `aifed` and `aifed-daemon`.

## Ecosystem

- [`aifed`](https://crates.io/crates/aifed) — the editor binary.
- [`aifed-daemon`](https://crates.io/crates/aifed-daemon) — the background daemon.
- [`aifed-daemon-client`](https://crates.io/crates/aifed-daemon-client) — Rust client for the daemon IPC protocol.

See the [top-level README](https://github.com/ImitationGameLabs/aifed#readme) for full documentation.
